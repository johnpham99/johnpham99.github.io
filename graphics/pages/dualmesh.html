Dual Mesh

Back to Index

This project is about modifying surface meshes.

In particular, this project reads in polyhedral model and creates a triangulated dual.

1-5: Changes the shape.
(tetrahedron, octahedron, icosahedron, star, torus)

d: Creates a triangulated dual of the current mesh.

n: Toggle between per-face and per-vertex normals.

r: Toggle between white and randomly colored faces.

Space: Toggle automatic rotation on and off.
